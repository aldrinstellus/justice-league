## Rollback Plan

### When to Rollback
**Automatic** (Immediate):
- Critical P0 incident
- Error rate > 50%
- All agents down
- Data corruption detected

**Manual** (Within 15 minutes):
- Error rate > 10%
- Performance degradation > 50%
- Multiple agents unhealthy
- Unresolved P1 incident

### Rollback Procedure
```bash
# Emergency rollback (< 5 minutes)
```
